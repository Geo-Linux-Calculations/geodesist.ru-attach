    1) Выпрямить этот скан, даже если он кажется ровным с помощью
       чего-нибудь вроде этого: https://geodesist.ru/threads/easy-trace.86416/
    2) Вставить выпрямленное изображение как подложку
    3) С помощью команды "_ALIGN" перенести это изображение на нужные координаты
    4) С помощью lisp-а:

    Код:
```
    ;;; Point with manual elevation input
    (defun c:pointh (/ pt1 h2)
        (setq pt1 (getpoint "\nSelect point: "))
        (setq h2 (getdist "\nElevation: "))
        (command
          "_POINT"
          (list
            (car pt1)
            (cadr pt1)
    ;        (caddr pt1)
            h2
          )
        )
        (princ)
    )
```

    5) По полученному облаку точек построить поверхность
