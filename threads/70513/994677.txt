        Специальный релиз для использования GeoEasy в качестве конвертера
        в GNU Gama ( https://geodesist.ru/threads/gnu-gama.86414/ ):

        https://github.com/Geo-Linux-Calculations/GeoEasy/releases/tag/3.1.4-gama

        1. Читается файл прибора (Sokkia, Leica, Topcon, Nikon) "Открыть..."
        2. Производится установка статуса "Грубых" для условных координат в "Правка > Координаты".
           (Если предполагается в GNU Gama выполнять 3D уравнивание, а высоты отсутствуют,
            то "Грубыми" назначаются все высоты, кроме одной).
        3. Производится предварительный расчёт "Вычисления > Пересчитать предварительные координаты".
           (Результат предварительного расчёта не особо важен,
            он производится для составления списка пунктов сети).
        4. Производится проверка статуса пунктов в "Правка > Координаты".
           Все исходные координаты не должны содержать "Грубые".
           (При 3D уравнивании высоты также должны быть назначены не "Грубыми"!).
        5. Производится экспорт сети "GNU Gama экспорт..." либо в 2D (*.g2d), либо в 3D (*.g3d).
           (Рекомендую подтверждать удаление предварительных координат в ходе экспорта).

        В ходе экспорта все углы будут пересчитаны из градусов в гоны.
        В случае 2D все наклонные расстояния будут пересчитаны в горизонтальные.

        GNU Gama частично входит в состав релиза. Формат комманды:
```
        Код:

        gama-local.exe you_file.g3d --language ru --svg you_file.svg --text you_file.log
```
